# Metric-Converter
This programme takes input from command line in order to convert the given multiple or submultiple unit of metric, into metric.
(eg. 1km = 1000m, 1mm = 0.001m etc)

Acceptable multiples:
T (Tera),
G (Giga),
M (Mega),
k (kilo),
m (meter),
d (deci),
c (centi),
m (milli),
u (micro),
n (nano),
p (pico)
